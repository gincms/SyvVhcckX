The Comparison Page

Comparing cryptocurrency market prices across exchanges

When using InterTrader, if you're not searching for profitable trades , a lot of your time will be spent on the comparison page, where you can see:

  • Prices of a cryptocurrency pair across multiple exchanges
  • Potential profit
  • Orderbooks from the exchanges where the coin is listed

Price chart

The prominent bar chart on the comparison page provides a quick overview of the buy and sell price across all the exchanges for which InterTrader has coin pair data.

Price analysis

Tapping bulb icon to the right of the chart displays the price analysis summary for the coin. This is the buy/sell price information described in the search and analysis guide. If there is potential profit to be earned, it is displayed here.

Exchange details

Below the bar chart are the numerical buy and sell prices for each exchange. You'll also find the coin name, where available, so you can more easily determine if the coins sharing the same ticker are being compared.

The quality of information returned from each exchange varies considerably. For this reason, the ticker symbol is the most reliable way to match and compare coins â€” but it isn't perfect. Make sure you verify that you're not trying to intertrade two different coins.

Orderbook

Tap the list icon for each exchange to see their orderbooks. InterTrader returns the last five buy and sell orders so that you can more quickly determine if there is enough supply and demand to make a profitable trade.
