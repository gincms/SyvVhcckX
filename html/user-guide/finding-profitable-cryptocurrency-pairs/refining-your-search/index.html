Refining your search

If you're not getting the results you want from the search page, whether too few or too many, there are a number of parameters you can change to refine your search and improve the results.

Tap the funnel icon in the top right corner of the page and it will reveal search form.

Max buy % difference

This is the difference between the lowest and highest buy across all the exchanges in InterTrader listing that trade pair. InterTrader defaults to a fairly liberal default of 500%. This is to ensure that the rare but genuine opportunities that have significant price differences across exchanges aren't missed.

Max sell % difference

This is the difference between the lowest and highest sell across all the exchanges in InterTrader listing that trade pair. InterTrader defaults to a fairly liberal default of 500%. This is to ensure that the rare but genuine opportunities that have significant price differences across exchanges aren't missed.

If you find you're getting a lot of false positives, try lowering the Max buy % difference and/or *Max sell % difference. Generally speaking, the higher these values, the more likely it is they're being skewed, either by wallet/exchange issues or coin-clash.

Min profit %

Only results that have at least the profit specified will be returned. The default is 3%.

Min no. of exchanges

Only trade pairs that are on at least this number of exchanges will be returned. The default minimum is 2.

Currency

You can limit the currency the coin is paired with if you're only interested in, for example, BTC pairs. By default, InterTrader includes all currencies.

Buy only at...

If you are only interested in opportunities to buy on specific exchanges, you can use this option to select them. Only trade pairs that have a cheap sell price on those exchanges will be returned. By default, all exchanges are included.

Sell only at...

If you are only interested in opportunities to sell on specific exchanges, you can use this option to select them. Only trade pairs that have a high buy price on those exchanges will be returned. By default, all exchanges are included.

Order by...

Results are ordered by Profit by default, but can be ordered by any of the following:

  • Profit
  • Avg. Buy price
  • Avg. Sell price
  • Symbol
  • Number of exchanges

Sort direction

Results can be sorted in descending (highest to lowest) or ascending (lowest to highest) value. The default sort direction is Descending.

Max no. of results

Limit the number of results by setting the maximum number of results. The default is 100.
