User Guide

How to use InterTrader to compare cryptocurrency exchange prices

InterTrader is designed to help you take advantage of the price variations between exchanges. This is a quick start guide with purely the essentials for you to use the app.

Getting started

Get the app

If you haven't already, download and install the app from Google Play.

Find some coins

When InterTrader starts you'll be on the Favourites page. This list will be empty, so start by tapping the Find some coins button .

Browse or search

Tap the exchange you'd like to browse. If looking for a specific coin, tap the search icon in the top right corner and start typing out the ticker symbol.

Find your coin

Browse the list of coins and find the pairing you're interested in. Where possible, InterTrader displays the name of the coins in the pair, as well as the ticker symbols, making it easier to differentiate between coins that share the same symbol. You can also add pairs directly to your Favourites by tapping the star in the list.

Compare exchange prices

Tap your preferred coin and you'll be taken to a screen similar to the one below. The green and red bars show the buy and sell prices, for each exchange named below them.

You can see the actual buy and sell figures in the boxes below the chart. LARGE, bold text means that you can potentially earn profit by buying on the exchange with the bold Sell price and selling on the exchnage will the bold Buy price.

To see potential profit, tap the Bitcoin icon. To get the latest prices, swipe down on the screen to pull and refresh it. If you'd like to visit the exchange to view the coin pair, just tap the box for that exchange.

Tap any of the icons to:

  • View profit info for the coin pair
  • Add the coin pair to your favourites
  • View the pair's orderbook for the specific exchange
